---
description: Discover Codex apps/connectors and summarize accessibility plus enabled status.
---

Run:

```bash
npm run apps:list
```

Then summarize:
- Total connectors found.
- Which are `accessible=true` and `enabled=true`.
- Any connectors that are accessible but disabled.
- Recommended connector mention syntax: `$<app-slug>` and `app://<id>`.
